Дані: Binance public API (без авторизації, прямі дзвінки з браузера). Basis = (Perp / Spot − 1) × 100. Funding річн. ≈ funding × 3 × 365. Сигнал з'являється коли |basis| ≥ поріг та обсяг ≥ мінімального.

Зелений рядок = сигнал у нормі. Жовтий = підвищена увага. Червоний = екстремум (|basis| ≥ 2× поріг). Не торгуй автоматично, перевіряй funding, ліквідність і новини.
